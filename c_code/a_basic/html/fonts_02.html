font관련 추가 사항

  • white-space : 의미없는 공백으로, 속성은 기본형인 normal, 공백제거인 nowrap, 보이는대로표시하는 pre, pre-wrap, pre-line
  • word-break : br요소의 디테일한 해석
  • text-overflow : 처리하는 내용으로, 위내용과 함께 처리
  • text-transform:대소문자 관리 uppercase, LOWERCASE
  • 글자의 크기설정 : base text subscript superscript small text
