내용을 작성할때에는, 때로 강조를 하고자 할 때도 있다.
굵은글씨를 사용하거나, 기울임쓰거나, 밑줄을 표기하거나, 취소선등을 넣고싶을때 코드를 작성하는 기능이 있지만,
위 내용은 모두 강조의 의미가 아니다.

웹문서에서 강조의 의미는 말 그대로 그 뜻을 전달가능해야 한다. 매우 중요 즉, 위험과 유사한 중요한 강조를 뜻하는 코드,
중요한 의미이지만, 강요보다는 권유의 성격띄고있는 코드,
강조 처리되는 기능으로 재확인의 의미, 기억의 의미,
이미 처리가되어 현재 내용을 더이상 사용할 수 없다 라는 의미를 기억해야하며,
i요소는 감정의 표현이라는 의미를 가지므로 주로, 아이콘 또는 이모티콘의 용도로 사용, s요소(strike)는 html5에서 완전히 삭제되어진 요소이므로 더이상 쓸 수 없다.

내용상 용어를 사용할 경우, 우리는 요약형태를 사용하는경우가 많다
